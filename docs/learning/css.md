## **CSS 基础选择器**

### **标签选择器（元素选择器）**

- **页面所有的同类标签**

### **类选择器**

- **class="类名"**
- **多个标签可以调用同一个类选择器**
- **一个标签可以同时调用多个类选择器**
- **类选择器命名规范**

  - **多个类名中间用空格隔开**
  - **不能以纯数字或者数字开头命名**
  - **不能出现特殊字符（除-\_连接符）**
  - **不建议中文**

### **id 选择器**

- **#id 名 { 属性 1:属性值 1; 属性 2:属性值 2; 属性 3:属性值 3; }**
- **同一个 id 选择器只能被调用一次，多次调用不符合 w3c 规范，js 调用出问题**
- **同一个标签不能调用多个 id 选择器，只能调用一下**

### **通配符选择器**

- **\* 代表所有，不推荐使用**

###

## **复合选择器**

### **交集选择器**

- **div.a {color: red;}**
- **必须满足两个条件，即是前边的标签，又调用后边的选择器（类选择器），中间没空格**

### **并集选择器**

- **.a,.b {color: red;}**
- **样式全部相同或者部分相同的选择器，通过逗号连在一起，进行集体声明（定义），最后一个选择器没有逗号**

### **后代选择器**

- **.a .b {color: red;}**
- **父元素在前，子元素在后，用空格相连**
- **后代选择器可以无限制隔代**
- **嵌套关系 选择父元素的后代（子元素）设置样式**
- **只要能代表父元素子元素，后代选择器可以是任意选择器的组合**

### **子元素选择器**

- **.a > .b {color: red;}**
- **选择父元素的直接下一代（亲儿子），用>连在一起，两边有空格**

### **相邻选择器**

- **.a + .b {color：red;}**
- **选择器 1（无效）+选择器 2（生效），是挨着的两个选择器，并列关系**

### **伪类选择器（内植类）**

- **链接伪类选择器**

  - **a:link 超链未访问的状态**

    - **:link 可以省略**

  - **a:visited 超链已访问的状态**
  - **a:hover 超链悬停的状态**
  - **a:active 超链激活的状态（按住鼠标不松手）**

- **都是利用交集选择器，任意标签都可以，查看注意清除浏览器缓存**

## **CSS 字体样式**

### **综合设置字体样式**

- **font: 倾斜 粗细 大小/行间距 字体;**

  - **必须按照顺序写，各个属性以空格隔开，可以省略（取默认值）**
  - **文字和大小必须设置，否则综合设置不生效**

### **字体风格 font-style**

- **normal 不倾斜（默认值）**
- **italic / oblique 斜体**

### **字体粗细 font-weight**

- **400 / normal 正常（默认值）**
- **700 / bold 加粗**
- **值范围在 100-900**

### **字号大小 font-size**

- **px 像素 相对长度单位**

### **字体 font-family**

- **默认 14px，最低识别 12px，使用偶数的数字字号**
- **中文字体加单引号，可多个字体，以英文逗号隔开，英文字体在前**
- **包含空格、#、$等符号，则该字体必须加英文状态下的单引号或双引号**
- **arial 英文字体**

### **字间距 letter-spacing**

- **letter-spacing: 10px;**
- **控制字(母)和字(母)之间的距离 值越大,间距越大,可以为负值**

### **词间距 word-spacing**

- **word-spacing: 20px;**
- **控制单词与单词之间的距离, 中文无效, 值越大,间距越大 可以为负值**

## **CSS 外观属性**

### **文本颜色 color**

- **十六进制，#开头，从 0-9，a-f，用六位来表示颜色**
- **RGB 颜色，rgb(255,0,0)或 rgb(100%,0%,0%）**

### **行间距 line-height**

- **normal 正常（默认值） 可以使用负值**
- **行高等于元素高度时，元素内部的文本会垂直居中**
- **通常行距比字号大 7.8 像素左右**
- **像素 px，相对值 em 和百分比%**

### **文本水平对齐方式 text-align**

- **left / center / right**
- 盒子里的文字 行内元素 行内块元素
- 需要父元素盒子有宽度

### **首行缩进 text-indent**

- **16px 默认浏览器文字大小**
- **缩两个单位，像素 px 或者相对值 em**

### **文本装饰 text-decoration **

- **none 默认定义标准的文本，取消下划线**
- **underline 定义文本下的一条线（下划线，链接自带的）**
- **overline 定义文本上的一条线**
- **line-through 定义穿过文本下的一条线**

### **文本方向 direction**

- **ltr 默认文本方向从左到右**
- **rtl 文本方向从右到左**
- **inherit 从父元素继承属性值**

## **元素显示模式**

### **块级元素（block-level）**

- **<h1>~<h6>、<div>、<p>、<ul>、<ol>、<li>、<dl>、<dt>、<dd>、<form>等**
- **1 独占一行或者多行
  2 默认高度是 0，会被内容撑开高度，宽度默认父元素宽度，不设置就是容器的 100%，居中一定要设宽度
  3 内外边距、高度、行高都可以控制
  4 是一个容器及盒子，里面可以放块、行内、行内块元素**
- **只有文字才能组成段落,因此 h1~h6，p，dt，他们都是文字类块级标签，里面不能放其他块级元素**

### **行内元素（inline-level）**

- **<a>、<span>、<strong>、<b>、<em>、<i>、<del>、<s>、<ins>、<u>、<sub>、<sup>等**
- **1 和相邻行内元素（行内块）在一行上,换行有空白缝隙
  2 默认宽高是 0，被内容撑开宽高，不能设置宽高
  3 左右内外边距、高度、行高都可以控制，上下无效
  4 只能放行内元素，块和行内块不可以放**

### **行内块元素（inline-block）**

- **<img />、<input />、<textarea>、<td>等**
- **1 和相邻行内元素（行内块）在一行上,换行有空白缝隙
  2 默认宽高就是它本身内容的宽度，可以设置宽高
  3 内外边距、高度、行高都可以控制
  4 只能放行内和行内块元素，块元素不可以放**

## **行高**

### **行高的测量**

- **默认浏览器文字大小 16px**
- **行高是基线与基线之间的距离**
- **normal 约等于 1.1-1.3**

  - **自测在 1.32 左右**
  - 淘宝 1.5

### **文字行高与盒子高度**

- **行高与容器高度一致 文字会垂直居中**
- **行高小于容器高度时 文字会偏上显示**
- **行高大于容器高度时 文字会偏下显示**

## **CSS 三大特性**

### **层叠性**

- **就近原则 哪个样式离着结构近，就执行那个样式**
- **多个样式作用于同一个（类）标签的时候，发生了样式冲突，权重相同的时候，后边的样式会覆盖（层叠）掉前边的样式，和样式调用顺序无关**

### **继承性 inherited**

- **[子承父业] 文字类子元素可以继承父元素的样式（text-，font-，line-这些元素开头的都可以继承，以及 color 属性）**
- **a 标签不能继承父元素的文字颜色**

  - **color: -webkit-link;**
  - **text-decoration: underline;**
  - **cursor: pointer;**

    - **鼠标小手**

- **p 标题标签不会继承父元素的文字大小**
- **继承的权重是 0，子元素优先自己的样式**

### **优先级**

- **继承/\* < 标签选择器/元素 < 类选择器/伪类 < id 选择器 < 行内样式 < ！important**
- **数位之间没有进制**
-

### **权重是优先级的算法，层叠是优先级的表现**

## **元素的显示与隐藏**

### **显示 display**

- **设置或检索对象是否及如何显示**
- **display: none; 隐藏对象
  display: block; 显示对象**

      - **隐藏之后，不再保留位置**

### **可见性 visibility**

- **设置或检索是否显示对象**
- **visibility: visible; 对象可视**

  - **隐藏之后，继续保留原有位置**

- **visibility: hidden; 对象隐藏**

  - **隐藏之后，继续保留原有位置**

### **溢出 overflow**

- **检索或设置当对象的内容超过其指定高度及宽度时如何管理内容**
- **overflow: visible; 默认值 溢出可见 不剪切内容也不添加滚动条**
- **overflow: auto; 超出自动显示滚动条，不超出不显示滚动条**
- **overflow: scroll; 不管超出内容否，总是显示滚动条**
- **overflow: hidden; 不显示超过对象尺寸的内容，超出的部分隐藏掉**

- normal 默认处理方式

### **文字溢出 text-overflow**

- **clip 不显示省略标记（...），而是简单的裁切**
- **ellipsis 当对象内文本溢出时显示省略标记（...）**

### **溢出文字省略替代隐藏**

  text-overflow: ellipsis; 文字用省略号替代超出的部分**
